# Initial Concept
Provide firmware and software for controlling Cephla's Squid microscope for various scientific imaging applications.

# Product Guide - Squid

## Goal
The primary goal of the Squid software is to provide a robust and flexible platform for controlling Cephla's Squid microscope. It aims to empower researchers with automated high-throughput imaging, manual control for exploratory work, and support for cutting-edge modalities like spatial omics and expansion microscopy, all while being extensible for custom hardware and algorithmic integration.

## Target Users
- **Academic Researchers and Scientists:** Utilizing advanced imaging for complex biological questions.
- **Lab Technicians and Imaging Core Staff:** Running routine, high-throughput imaging protocols.
- **Developers:** Building and integrating custom imaging pipelines and hardware extensions.

## Core Goals
- **High-Throughput Automation:** Reliable automated slide scanning and multi-well screening.
- **Intuitive Manual Control:** User-friendly interface for real-time microscope interaction and live imaging.
- **Advanced Modality Support:** Specialized workflows for spatial omics, expansion microscopy, and fluidics integration.
- **Extensible Platform:** Flexible architecture for integrating new hardware and custom analysis algorithms.

## Key Features
- **Automated Acquisition:** Precise control over multi-point and multi-channel imaging sequences.
- **Real-time Visualization:** High-performance image processing and display for immediate feedback.
- **Hardware Synchronization:** Tight orchestration of lasers, stages, cameras, and fluidics systems.

## Focus Areas
- **Stability and Performance:** Ensuring robust operation for long-duration automated acquisitions.
- **UX/UI Enhancement:** Continual improvement of the user interface for better usability and accessibility across all user types.
