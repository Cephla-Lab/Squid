## Context

arch_v2's X-Light driver (`backend/drivers/lighting/xlight.py`) currently assumes V3/Cicero protocol (115200 baud) and uses the `idc` command for configuration detection. V1/V2 devices use 9600 baud and don't support the `idc` command, so they fail to connect.
